Navigating the Data Lake

EDITO uses a SpatioTemporal Asset Catalog (STAC).
The STAC endpoint is https://catalog.dive.edito.eu and can be viewed on EDITO datalab > integrate data or queried programmatically.

EMODNET-Biology Data (Cluster 1 and 2)

All EMODnet biology data is made available in this STAC collection:
https://api.dive.edito.eu/data/collections/emodnet-occurrence_data/

At the moment of this visualization it contains this parquet file:
https://s3.waw3-1.cloudferro.com/emodnet/emodnet_biology/12639/marine_biodiversity_observations_2026-02-26.parquet

The file was filtered to select the matching datasets by their dasid. A list of all datasets and previews is available below. For the occurrence extraction each unique lat, lon, time & aphia was considered.

ETN Data (Cluster 3)

Data is stored in this collection:
https://www.lifewatch.be/etn/parquet/animal_tracking_datasets/collection.json
