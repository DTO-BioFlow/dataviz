    Data flows

    The DTO-Bioflow project contributes data to EDITO from diverse sources and via several pipelines.
    The full description of data flows can be consuled at the project website (https://dto-bioflow.eu/data-flow) and via the blueprint (https://zenodo.org/records/16538490)

    Data visualization overview

    Datasets are clustered in groups for visualization purposes.
    • Data submitted to EMODnet and ingested in EDITO via pre-existing EMODnet-EDITO pipeline
      • Work package 2 observation data (call-1)
      • Work package 3 sensor observation data
    • Data hosted on other databases and ingested in EDTIO via newly developed pipelines
      • ETN data (European Tracking Network)
