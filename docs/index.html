DTO-Bioflow Data Contribution to EDITO: Visualization

Data flows

The DTO-Bioflow project contributes data to EDITO from diverse sources and via several pipelines.
The full description of data flows can be consuled at the project website (https://dto-bioflow.eu/data-flow) and via the blueprint (https://zenodo.org/records/16538490).

This flowchart below illustrates the main concept.
flowchart TD A[Data] --> B[EMODnet-Biology] B -- [pre-existing pipeline] --> D[EDITO Infra data lake] C[data] -- [direct ingestion] --> D

Data Clusters

Datasets can be clustered depending on their source and pipeline.
Following clusters have been identified:
  • EMODnet Pipeline: data submitted to EMODnet and ingested in EDITO via pre-existing EMODnet-EDITO pipeline.
    • Cluster 1: New Observation data from open call for data providers (with FSTP grants)
    • Cluster 2: Sensor observation data
  • Direct connections: data hosted on other databases and ingested in EDITO via newly developed pipelines.
    • Cluster 3: Sensor data too extensive for EMODnet-Biology (ETN: European Tracking Network)
