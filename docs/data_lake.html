Data Lake

STAC

EDITO uses a SpatioTemporal Asset Catalog (STAC).
The STAC endpoint https://catalog.dive.edito.eu. can be viewed on EDITO datalab > integrate data or queried programmatically.

Cluster 1 and 2: EMODNET-Biology

All EMODnet biology data is made available in this collection:
https://api.dive.edito.eu/data/collections/emodnet-occurrence_data/
which at this moment contains this parquet file:
https://s3.waw3-1.cloudferro.com/emodnet/emodnet_biology/12639/marine_biodiversity_observations_2026-02-26.parquet
The file was filtered to select the matching datasets by their dasid.

Cluster 3: ETN

Data is stored in this collection:
https://www.lifewatch.be/etn/parquet/animal_tracking_datasets/collection.json
